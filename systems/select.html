Components / Form Controls

Select

Select is a native single-value control for short, simple lists where browser and mobile option behavior is acceptable.

SelectionKnown optionsSingle value

Overview

Use Select when the option set is known, short enough to scan, and native browser option presentation is acceptable. Use Combobox when the list is searchable, requires filtering, or needs controlled option styling.

Interactive Example

Select uses the same field label, helper, error, hover, and focus treatment as text-entry controls.

IEP Details

Choose the type that matches the current process.

Behavior

Select should provide a clear placeholder, preserve the selected value, and show local validation when a required selection is missing. The closed field surface uses shared field styling; the open menu and option rows remain browser controlled.

Use Select When

The option list is short enough to scan, and native browser/mobile menu behavior is acceptable.

Use Combobox Instead

The option list is long, searchable, needs designed option rows, or supports adding a new item.
