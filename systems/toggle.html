Toggle

Toggle is a boolean control for immediate on/off state changes. Use it when the selected state can be understood without comparing multiple options.

PLANNED Boolean Immediate state On / off

Overview

This page is planned. Guidance and examples are not yet final. Toggle deserves its own page because it behaves differently from checkboxes. It usually changes a setting immediately and must make the current state clear.

Behavior
