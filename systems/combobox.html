Combobox

Combobox combines text input with a controlled option list. Use it when users need to search, filter, or select from an option set whose open menu and option rows need system styling.

Searchable selectionFilteringOptional create-new

Overview

Combobox is the controlled-menu selection pattern. The user types to narrow a constrained list, then commits one of the available values unless create-new behavior is explicitly supported.

Combobox vs Select

Use Combobox when the list is too long to scan, users benefit from filtering, or the option menu needs consistent system styling.

Combobox vs Text Input

Use Text Input when the value is unconstrained and users do not select from system-defined options.
