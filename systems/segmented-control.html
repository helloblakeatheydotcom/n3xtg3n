Segmented Control

Segmented Control is a compact selection pattern for short, mutually exclusive states. Use it when the options are peers, the set has two to four options, and the selected state should remain visible inside a form or dense row.

Selection Radio-backed Mutually exclusive Form control

Overview

Segmented Control belongs with selection controls, but it is visually denser than a standard radio group. Use it for explicit state or mode choices with short labels. Prefer two or three options, and cap the pattern at four options.

Interactive Example

The default form version uses a radio group so the value submits like a normal single-select field.

Behavior

Use segmented controls for short, stable option sets with two or more choices. Do not exceed four options; if the labels are long, the choice needs helper text per option, or the list grows beyond four, use radio buttons, radio cards, or a select instead.
