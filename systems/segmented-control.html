Segmented Control

Segmented Control is a compact selection pattern for mutually exclusive options, usually used for modes, filters, or view switches where all options should remain visible.

PLANNED Selection Mutually exclusive Compact options

Overview

This page is planned. Guidance and examples are not yet final. Segmented Control belongs with selection controls but should be documented separately from radio groups because its visual density, interaction pattern, and placement rules are different.

Interactive Example

Click an option to change the selected segment.

View Options

Behavior

Use segmented controls for short, stable option sets. Use radio groups when labels or descriptions need more space, or when the choice is part of a larger form question.
