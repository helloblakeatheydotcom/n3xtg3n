Text Area

Text Area is the multi-line control for longer direct-entry content. Use it when users need room to write narrative, notes, descriptions, or explanations.

Multi-line entryNarrative contentShared field states

Overview

Text Area uses the same label, helper text, validation, and state language as Text Input, but provides a taller writing surface for content that cannot be captured comfortably in one line.

Use Text Area When

The value is narrativeParent input, notes, rationales, descriptions, and explanations are good fits.

The length is variableUse Text Area when the user may need more room than a single-line input provides.

Sizing

Default heightThe base control uses the shared textarea minimum height.

Layout spanText areas often use span 6 or span 12 depending on expected content length.
