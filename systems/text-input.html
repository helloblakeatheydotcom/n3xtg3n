Text Input

Text Input is the base single-line control for direct text entry. Use it for names, short descriptions, identifiers, and other values that do not require a specialized picker or selection pattern.

Single-line entryInteractive validationShared field tokens

Overview

Text Input defines the shared anatomy, states, helper text behavior, validation behavior, and layout expectations used by single-line entry controls and many formatted input variants.

Use Text Input When

The value is known by the userNames, IDs, short descriptions, and other direct-entry values are good fits.

The value is not selected from a listUse Select or Combobox when users choose from constrained options.

The entry is one lineUse Text Area when the value needs multiple lines or narrative content.

Layout

Standard widthStandalone text inputs typically use span 4 in the shared layout system.

Longer valuesUse span 6 or span 12 when labels, helper text, or value length need more room.

Panel-containedText inputs should appear inside panels on process pages.
