Components / Inputs

Input systems for text entry and selection.

Inputs cover structured entry, grouped choices, and future compact input controls. Use this section to distinguish text-based fields from selection-based controls before choosing the specific pattern.

Layout Relationship

Fields participate in the shared layout system rather than defining their own widths in isolation.

Field Spans

Fields support spans of 3, 6, 9, and 12 within the shared 12-column layout system. Use span 6 by default, then use 3 for short structured pairs, 9 for long inputs, and 12 for full-width content.

Foundation Reference

Use the Layout foundation for the 12-column grid, 16px gutters, panel container width, and responsive 4 / 2 / 1 column behavior that input layouts follow.

See Foundations / Layout for the shared grid, gutters, breakpoints, containers, and span system.
