Foundation

Color

The NextGen color system works in three tiers: primitive scales, semantic tokens, and component tokens. Primitive scales define the raw color families, semantic tokens define what those colors mean, and component tokens document how individual components apply them. Component-level tokens are documented on each component's own Anatomy & Specs tab.

How This System Works

This page is organized into primitive scales, semantic tokens, and component-level documentation. Primitive scales show the raw families, semantic tokens show meaning, and component tokens live with each component’s own Anatomy & Specs page.

Primitive Families

These rows show the approved primitive families for the system, including active component colors and reserved future stops. Scales are intentionally sparse and ordered from lightest to darkest.
